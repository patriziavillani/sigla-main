===============
Coda File Excel
===============

In tutte le funzioni di consultazione (o quelle di gestione in cui è prevista la ricerca con risultati in griglia) è possibile effettuare stampe (creazione di un file pdf) oppure estrazione dei dati in un file excel. 

Le griglie, dopo l'esecuzione delle ricerche, si presentano nel modo seguente:

![](screenshot/map_6.png)

E' possibile quindi produrre le stampe:

![](screenshot/print_4.png)

Che saranno poi riportate nella coda di stampe.
